Projeto - Jogo da Velha Inteligente

Enunciado: neste projeto vocês deverão utilizar o arquivo base fornecido nesse repositório, que se trata de um jogo da velha para jogador vs. máquina, e preencher as funções da jogada da máquina! Devem ser mapeados os casos posíveis e utilizando if's (MUUUITOS if's) desenvolvam um algoritmo capaz de nunca perder (no pior dos casos empatar!)

Sintam-se livres para alterar o código fornecido como bem entenderem, não se limitem a mexer apenas na função a ser preenchida, a base é apenas para nortear vocês!
